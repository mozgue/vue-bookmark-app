# Vue Bookmark App

## Start scaffold

`npm create vue@latest`

## Features

- Vite + Vue 3 Composition API
- TypeScript
- Pinia (state management)
- Axios (http-клиент)
- Router (SPA development) \*
- ESLint (error prevention)
- Prettier (code formatting)

\* - _будут добавлены вручную во время разработки_

### Первый запуск

<pre>
npm i
npm run format
npm run dev
</pre>
